---
title: 项目审核
description: This guide provides instructions on customizing our application.
---

开源项目需上会评审后才能发布任务。
技术委员会通过会议讨论，对项目进行审核。审核周期为任务提交后 5~10 个工作日内。


1. 开源项目提交注册资料后，秘书处将对资料进行初步审核，查验相关信息是否完备和属实，并将项目按照技术领域归类。
2. 通过初审的项目，将由秘书处通知准备评审材料。
3. 项目负责人按照约定时间到技术委员会进行陈述，同委员会进行交流。
4. 委员会会后决策项目是否通过评审，未通过的项目告知意见或修改建议。
5. 通过评审的项目可自行添加导师，然后发布任务；导师必须是项目Maintainer；发布任务需满足要求，具体要求参考下文。
