要使用 Bot 命令，需要在对应的任务下面回复命令

### 学生命令

| 命令名称          | 命令说明                                                 |
| ----------------- | -------------------------------------------------------- |
| /request-assign   | 学生认领一个 **Open** 状态的任务                              |
| /reuqets-complete | 学生完成 PR 后申请完成任务，任务将从 **已认领** 变为 **审核** 状态 |
| /request-release  | 学生放弃任务，任务将变为 **Ope**n                             |

### 导师命令

| 命令名称           | 命令说明                                             |
| ------------------ | ---------------------------------------------------- |
| /intern-approve    | 导师同意认领，任务将从认领审核变为已认领             |
| /intern-disapprove | 导师拒绝认领，任务将从认领审核变为 **Open**               |
| /intern-fail       | 导师合入审核失败，终止该学生的任务，任务将变为 **Open** |
| /intern-done       | 导师合入审核成功，任务完成关闭，发放积分             |
| /intern-close      | 导师直接关闭任务，适用于分数 **label** 填写错误的情况      |
