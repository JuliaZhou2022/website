## 什么样的项目可以加入 R2CN 开源实习计划？

1. 项目必须是开源，且代码仓库托管在 GitHub 上；
2. 项目必须是使用 Rust 编程语言开发；
3. 项目必须是云原生技术相关领域；

以下是 R2CN 开源实习计划的开源项目技术栈示意图，可以根据此示意图来判断自己的项目是否符合 R2CN 的要求。

<img src="/images/project/signup/r2cn-stack.png" />

如果开源项目对自己是否符合 R2CN 的技术栈范围有疑问，可以和 [@genedna](https://github.com/genedna) 联系，讨论请发送邮件到 `eli@patch.sh` 或在 [论坛](https://github.com/orgs/r2cn-dev/discussions) 发起讨论咨询。

## 开源项目自行在官网进行注册：
（1）填写项目联系人、技术领域、项目仓库地址等信息；<br/>
（2）同时签署相关协议。
