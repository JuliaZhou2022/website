---
title: 任务 Issue 标签
description: R2CN Bot 使用 Issue 标签追踪所有 R2CN 任务
---

<Note type="danger" title="任务 Issue 标签">
    1. [R2CN Bot](/docs/project/bot) 会监听标签事件，新建 Issue 任务时需要创建 `r2cn-x` 标签，`x` 为任务分值，如： `r2cn-2` 表示该任务为 `2` 分；
    2. 任务创建后，随着任务的执行可能会遇到工作量大于或小于预期，可以通过更换分值标签来更新分值；分值不能超过系统授权的分值，超过修改将失败；
    3. 建议开源项目在仓库的 Issue 预先设定多个 `r2cn-x` 标签，方便导师快速选择任务分值；
</Note>
