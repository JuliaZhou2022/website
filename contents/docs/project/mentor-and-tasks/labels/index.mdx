### 添加标签
- R2CN Bot会监听标签事件，只有添加`r2cn-x` 标签才能新建R2CN任务
- 标签后需要添加任务的分值 如： `r2cn-10` 表示该任务为`10`分
- 任务一旦创建便不能够修改分数，如需要修改积分，请先使用`intern-close`命令关闭任务追踪，再重新附加标签

### 分值统计
1. 在代码托管平台为相应项目部署webhook，统计学生和导师的动作。
2. 将webhook获得的数据存储在云数据库中。
3. 每月底对数据库中的数据进行统计，得出学生的积分情况，算出学生奖金并导出表格。
4. 运营人员审核积分和奖金的情况
