---
title: 任务分值规则
description: 任务分值规则是导师发布实习任务时设定任务的参考标准，根据任务的难易程度和所需时间来确定任务的分值。
---

| 任务分值 | 难易程度 | 所需时间   |
| -------- | -------- | -------- |
| 2        | 简单     | ≤ 8h       |
| 5        | 简单     | 8 ~ 20h    |
| 10       | 简单     | 21 ~ 40h   |
| 20       | 中等     | 41 ~ 80h   |
| 30       | 中等     | 81 ~ 120h  |
| 40       | 难       | 121 ~ 160h |
| 50       | 难       | ≥ 161h     |
